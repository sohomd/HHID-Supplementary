# HHID GitHub Pages Starter

This is a ready-to-deploy static GitHub Pages starter for the HHID dataset website.

## Folder structure

- `index.html` — main landing page
- `styles.css` — page styling
- `assets/images/` — figures, posters, thumbnails
- `assets/videos/` — teaser and qualitative videos
- `docs/` — paper PDF and extra downloadable files

## How to deploy on GitHub Pages

1. Create a new GitHub repository, for example `HHID`.
2. Upload all files from this folder to the repository root.
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
5. Save. GitHub will publish the site in a minute or two.

## What to replace next

- Replace placeholder text in `index.html`
- Add your paper PDF to `docs/HHID_paper.pdf`
- Add your figures to `assets/images/`
- Add teaser videos to `assets/videos/`
- Replace the citation block
- Add your real repository and dataset links

## Suggested next step

After deployment, update the paper link in your ACM MM submission website field with the GitHub Pages URL.
